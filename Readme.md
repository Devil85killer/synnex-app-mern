## 📌 Project Overview
**Synnex** is a comprehensive, scalable, and highly secure MERN-stack web application designed to bridge the gap between students, alumni, and university administration. It serves as a centralized hub for networking, event management, career opportunities, and campus updates.

Built with modern web technologies, Synnex features a robust Role-Based Access Control (RBAC) system, ensuring customized experiences for Students, Alumni, Teachers, and Admins.

---

## ✨ Key Features
- **🔐 Secure Authentication:** JWT-based login with encrypted passwords (Bcrypt) and secure password reset capabilities.
- **📊 Advanced Admin Dashboard:** A powerful control panel to manage users, approve jobs, publish news, and oversee platform activities.
- **📅 Dynamic Event Management:** Seamless event creation, one-click user registration, and real-time attendee tracking with auto-data healing.
- **💼 Career & Job Board:** Dedicated space for alumni to post job opportunities and students to explore them.
- **📰 Notice Board & Feedback:** Real-time news broadcasts and a dedicated feedback mechanism for continuous platform improvement.
- **🛡️ Bulletproof Security:** CORS configuration, MongoDB Injection protection, and strict API route guarding.

---

## 🛠️ Technology Stack
* **Frontend:** React.js, Tailwind CSS, Heroicons, React Router DOM, Axios, React-Toastify
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Atlas), Mongoose (ODM)
* **Security & Auth:** JSON Web Tokens (JWT), Cookie-Parser, Bcrypt.js
* **Deployment:** Vercel (Frontend), Render (Backend)
